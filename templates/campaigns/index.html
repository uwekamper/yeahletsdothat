Yeah Let's Do That

Yeah Let's Do That is a crowd-funding platform for small user-groups (e.g. non-profits, maker-spaces, and hacker-spaces).

It helps those groups to finance their endeavours together. Yeah Let's Do That is open-source and it runs on your own server (hosted options might by available in the future). You can see the code on Github and there is also an Android app available.
